Guidlines for using this site

  • In order to sign in as admin, use the mongo shell and set the admin flag on username to true

  • open up a third node cmd
  • move to the Confusion data base use Confusion
  • show all the users > db.users.find().pretty()
  • copy the username line and put it in the new query find({"username" : "jeff11"}) to see the status of the admin flag
  • use the UPDATE methode with $SET to change the admin db.users.update({ "username" : "jeff11"},{$set: { "admin": true}})
  • restart the app

  • Logging in to the site - use Postman

  • open Postman and input localhost:3443
  • in the body write all your details {"username":"jeff11", "password":"1234", "firstname":"jeff:,"lastname":"levy"}
  • post using /register or login (if you have already registered)
  • copy the token without the "" and past inside the header, with key x-access-token
